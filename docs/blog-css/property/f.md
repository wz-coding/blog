以 F 为首的 CSS 属性(样式)

# F

#### 以 F 为首的属性

<Mcard>

### [`fallback`][en-link]{ .mdn-link linkparam=@counter-style/fallback}
如果当前计数器样式（`@counter-style`）无法为特定计数器值创建标记表示，回退描述符可用于指定回退到的计数器样式
</Mcard>

<Mcard>

### [`filter`][zh-link]{ .mdn-link linkparam=filter}
将模糊或颜色偏移等图形效果应用于元素。通常用于调整图像、背景和边框的渲染
</Mcard>

<Mcard>

### [`<filter-function>`][zh-link]{ .mdn-link linkparam=filter-function}
CSS 数据类型，代表可以改变输入图像外观的图形效果，它可以用于 `filter` 和 `backdrop-filter` 属性
</Mcard>

<Mcard>

### [`:first`][zh-link]{ .mdn-link linkparam=:first}
CSS 伪类，与 `@page` at 规则一起使用。表示打印文档的第一页
</Mcard>

<Mcard>

### [`:first-child`][zh-link]{ .mdn-link linkparam=:first-child}
CSS 伪类，表示在一组兄弟元素中的第一个元素
</Mcard>

<Mcard>

### [`::first-letter(:first-letter)`][zh-link]{ .mdn-link linkparam=::first-letter(:first-letter)}
CSS 伪元素，选中块级元素第一行的第一个字母，并且文字所处的行之前没有其他内容（如图片和内联的表格）
</Mcard>

<Mcard>

### [`::first-line(:first-line)`][zh-link]{ .mdn-link linkparam=::first-line(:first-line)}
CSS 伪元素，设置块级元素的第一行应用的样式
</Mcard>

<Mcard>

### [`:first-of-type`][zh-link]{ .mdn-link linkparam=:first-of-type}
CSS 伪类，表示一组兄弟元素中其类型的第一个元素
</Mcard>

<Mcard>

### [`fit-content`][zh-link]{ .mdn-link linkparam=fit-content}
设置元素的 `width`、`height`、`min-width`、`min-height`、`max-width` 和 `max-height`，此时最大和最小尺寸将基于元素的内容尺寸计算
</Mcard>

<Mcard>

### [`<flex>`][zh-link]{ .mdn-link linkparam=flex}
CSS 数据类型，表示了网格（grid）容器中的一段可变长度。于 `grid-template-columns`、`grid-template-rows` 及相关属性中使用
</Mcard>

<Mcard>

### [`flex`][zh-link]{ .mdn-link linkparam=flex}
CSS 简写属性，是 `flex-grow`，`flex-shrink`，`flex-basis` 的简写，设置了弹性项目如何增大或缩小以适应其弹性容器中可用的空间
</Mcard>

<Mcard>

### [`flex-basis`][zh-link]{ .mdn-link linkparam=flex-basis}
指定了 flex 元素在主轴方向上的初始大小。如果不使用 `box-sizing` 改变盒模型的话，那么这个属性就决定了 flex 元素的内容盒 `content-box` 的尺寸
</Mcard>

<Mcard>

### [`flex-direction`][zh-link]{ .mdn-link linkparam=flex-direction}
指定了内部元素是如何在 flex 容器中布局的，定义了主轴的方向
</Mcard>

<Mcard>

### [`flex-flow`][zh-link]{ .mdn-link linkparam=flex-flow}
CSS 简写属性，是 `flex-direction` 和 `flex-wrap` 的简写
</Mcard>

<Mcard>

### [`flex-grow`][zh-link]{ .mdn-link linkparam=flex-grow}
设置 flex 项 主尺寸 的 flex 增长系数
</Mcard>

<Mcard>

### [`flex-shrink`][zh-link]{ .mdn-link linkparam=flex-shrink}
指定了 flex 元素的收缩规则。flex 元素仅在默认宽度之和大于容器的时候才会发生收缩，其收缩的大小是依据 `flex-shrink` 的值
</Mcard>

<Mcard>

### [`flex-wrap`][zh-link]{ .mdn-link linkparam=flex-wrap}
指定 flex 元素单行显示还是多行显示。如果允许换行，这个属性允许你控制行的堆叠方向
</Mcard>

<Mcard>

### [`fr`][zh-link]{ .mdn-link linkparam=flex_value#fr}
CSS数据类型，flex 布局的单位，`number` 后加单位 `fr`
</Mcard>

<Mcard>

### [`float`][zh-link]{ .mdn-link linkparam=float}
指定一个元素应沿其容器的左侧或右侧放置，允许文本和内联元素环绕它。该元素从网页的正常流动（文档流）中移除，但是仍然保持部分的流动性（与绝对定位相反）
</Mcard>

<Mcard>

### [`:focus`][zh-link]{ .mdn-link linkparam=:focus}
CSS 伪类，表示获得焦点的元素
</Mcard>

<Mcard>

### [`:focus-visible`][zh-link]{ .mdn-link linkparam=:focus-visible}
CSS 伪类，当元素匹配 `:focus` 伪类并且客户端 (UA) 的启发式引擎决定焦点应当可见时，`:focus-visible` 伪类将生效
</Mcard>

<Mcard>

### [`:focus-within`][zh-link]{ .mdn-link linkparam=:focus-within}
CSS 伪类，表示当元素或其任意后代元素被聚焦时，将匹配该元素
</Mcard>

<Mcard>

### [`font`][zh-link]{ .mdn-link linkparam=font}
CSS 简写属性，是 `font-style`, `font-variant`, `font-weight`, `font-size`, `line-height` 和 `font-family` 属性的简写，可以将元素的字体设置为系统字体
</Mcard>

<Mcard>

### [`font-display`][zh-link]{ .mdn-link linkparam=@font-face/font-display}
决定了一个 @font-face 在不同的下载时间和可用时间下是如何展示的
</Mcard>

<Mcard>

### [`@font-face`][zh-link]{ .mdn-link linkparam=@font-face}
CSS at-rule，指定一个用于显示文本的自定义字体；字体能从远程服务器或者用户本地安装的字体加载
</Mcard>

<Mcard>

### [`font-family`][zh-link]{ .mdn-link linkparam=font-family}
通过给定一个有先后顺序的，由字体名或者字体族名组成的列表来为选定的元素设置字体
</Mcard>

<Mcard>

### [`font-family(@font-face)`][zh-link]{ .mdn-link linkparam=@font-face/font-family}
为页面上的字体指定一个由 `@font-face` 规则定义的字体族
</Mcard>

<Mcard>

### [`font-feature-settings`][zh-link]{ .mdn-link linkparam=font-feature-settings}
用于控制 OpenType 字体中的高级印刷功能
</Mcard>

<Mcard>

### [`@font-feature-values`][zh-link]{ .mdn-link linkparam=@font-feature-values}
CSS at-rule，允许作者在 `font-variant-alternates` 中使用通用名称，用于在 OpenType 中以不同方式激活功能。它允许在使用几种字体时简化 CSS
</Mcard>

<Mcard>

### [`font-kerning`][zh-link]{ .mdn-link linkparam=font-kerning}
设置是否使用字体中储存的字距信息
</Mcard>

<Mcard>

### [`font-language-override`][zh-link]{ .mdn-link linkparam=font-language-override}
控制在指定的语言中使用特定的字形
</Mcard>

<Mcard>

### [`font-optical-sizing`][en-link]{ .mdn-link linkparam=font-optical-sizing}
设置文本渲染是否针对不同尺寸的视图进行优化调整
</Mcard>

<Mcard>

### [`font-size`][zh-link]{ .mdn-link linkparam=font-size}
设置字体大小。更改字体大小还会更新字体大小相关的 `length` 单位，例如 `em`、`ex` 等
</Mcard>

<Mcard>

### [`font-size-adjust`][zh-link]{ .mdn-link linkparam=font-size-adjust}
定义字体大小应取决于小写字母，而不是大写字母。在字体较小时，字体的可读性主要由小写字母的大小决定，通过此选项即可进行调整
</Mcard>

<Mcard>

### [`font-stretch`][zh-link]{ .mdn-link linkparam=font-stretch}
为字体定义一个正常或经过伸缩变形的字体外观，这个属性并不会通过伸展/缩小而改变字体的几何外形
</Mcard>

<Mcard>

### [`font-style`][zh-link]{ .mdn-link linkparam=font-style}
设置 `font-family` 字体下的 `italic` 或 `oblique` 样式
</Mcard>

<Mcard>

### [`font-synthesis`][zh-link]{ .mdn-link linkparam=font-synthesis}
控制浏览器可以合成（synthesize）哪些缺失的字体，粗体或斜体
</Mcard>

<Mcard>

### [`font-variant`][zh-link]{ .mdn-link linkparam=font-variant}
CSS 简写属性，是 `font-variant-caps`, `font-variant-numeric`, `font-variant-alternates`, `font-variant-ligatures`, `font-variant-east-asian` 等属性的简写
</Mcard>

<Mcard>

### [`font-variant-alternates`][zh-link]{ .mdn-link linkparam=font-variant-alternates}
控制备用字体的使用，这些替代字体可以通过 `@font-feature-values` 中定义的替代名称来引用
</Mcard>

<Mcard>

### [`font-variant-caps`][zh-link]{ .mdn-link linkparam=font-variant-caps}
控制大写字母特殊字符的使用
</Mcard>

<Mcard>

### [`font-variant-east-asian`][en-link]{ .mdn-link linkparam=font-variant-east-asian}
控制东亚文字（如日语和中文）的替代字形的使用
</Mcard>

<Mcard>

### [`font-variant-ligatures`][zh-link]{ .mdn-link linkparam=font-variant-ligatures}
控制其所应用元素文本的 `ligatures` 与 `contextual forms` 。会使文字最终的表现形式更加统一
</Mcard>

<Mcard>

### [`font-variant-numeric`][zh-link]{ .mdn-link linkparam=font-variant-numeric}
控制数字、分数和序号标记的替代字形的使用
</Mcard>

<Mcard>

### [`font-variant-position`][zh-link]{ .mdn-link linkparam=font-variant-position}
控制替代的、较小的字形的使用，这些字形定位为上标或下标
</Mcard>

<Mcard>

### [`font-variation-settings`][zh-link]{ .mdn-link linkparam=font-variation-settings}
提供了对 可变字体(variable font) 特征的低级控制，通过指定要更改的特征的四个字母轴名称及其值
</Mcard>

<Mcard>

### [`font-weight`][zh-link]{ .mdn-link linkparam=font-weight}
指定字体的粗细程度
</Mcard>

<Mcard>

### [`forced-color-adjust`][en-link]{ .mdn-link linkparam=forced-color-adjust}
选择某些元素退出强制颜色模式。然后将这些值的控制权恢复到 CSS
</Mcard>

<Mcard>

### [`format()`][zh-link]{ .mdn-link linkparam=@font-face/src#format()}
`@font-face` 中的 src 描述符指定了包含字体数据的资源。`@font-face` 规则中需要指定这个属性
</Mcard>

<Mcard>

### [`<frequency>`][zh-link]{ .mdn-link linkparam=frequency}
CSS数据类型，表示频率维度，例如语音的音高。目前它未在任何 CSS 属性中被使用
</Mcard>

<Mcard>

### [`:fullscreen`][zh-link]{ .mdn-link linkparam=:fullscreen}
CSS 伪类，匹配当前处于全屏模式的所有元素
</Mcard>

<Mcard>

### [`:future`][en-link]{ .mdn-link linkparam=:future}
CSS 伪类，匹配任何完全出现在与 `:current` 匹配的元素之后的元素。例如，在带有由 WebVTT 显示的字幕的视频中
</Mcard>
