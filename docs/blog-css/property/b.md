# B

#### 以 B 为首的属性

<Mcard>

### [`::backdrop`][zh-link]{ .mdn-link linkparam=::backdrop}
伪元素，在任何处于全屏模式的元素下的即刻渲染的盒子
</Mcard>

<Mcard>

### [`backdrop-filter`][zh-link]{ .mdn-link linkparam=backdrop-filter}
为一个元素后面区域添加图形效果（如模糊或颜色偏移）。因为它适用于元素背后的所有元素，为了看到效果，必须使元素或其背景至少部分透明
</Mcard>

<Mcard>

### [`backface-visibility`][zh-link]{ .mdn-link linkparam=backface-visibility}
指定当元素背面朝向观察者时是否可见
</Mcard>

<Mcard>

### [`background`][zh-link]{ .mdn-link linkparam=background}
CSS 简写属性，用于一次性集中定义各种背景属性
</Mcard>

<Mcard>

### [`background-attachment`][zh-link]{ .mdn-link linkparam=background-attachment}
决定背景图像的位置是在视口内固定，或者随着包含它的区块滚动
</Mcard>

<Mcard>

### [`background-blend-mode`][zh-link]{ .mdn-link linkparam=background-blend-mode}
定义该元素的背景图片，以及背景色如何混合
</Mcard>

<Mcard>

### [`background-clip`][zh-link]{ .mdn-link linkparam=background-clip}
设置元素的背景（背景图片或颜色）是否延伸到边框、内边距盒子、内容盒子下面
</Mcard>

<Mcard>

### [`background-color`][zh-link]{ .mdn-link linkparam=background-color}
设置元素的背景色，属性的值为颜色值或关键字 `transparent` 二者选其一
</Mcard>

<Mcard>

### [`background-image`][zh-link]{ .mdn-link linkparam=background-image}
为一个元素设置一个或者多个背景图像
</Mcard>

<Mcard>

### [`background-origin`][zh-link]{ .mdn-link linkparam=background-origin}
规定了指定背景图片 `background-image` 属性的原点位置的背景相对区域
</Mcard>

<Mcard>

### [`background-position`][zh-link]{ .mdn-link linkparam=background-position}
为每一个背景图片设置初始位置。这个位置是相对于由 `background-origin` 定义的位置图层的
</Mcard>

<Mcard>

### [`background-position-x`][zh-link]{ .mdn-link linkparam=background-position-x}
设置水平方向的位置，与每个背景图片等位置层设置属性 `background-origin` 相关
</Mcard>

<Mcard>

### [`background-position-y`][zh-link]{ .mdn-link linkparam=background-position-y}
设置初始状态时背景图片在垂直方向上的位置，这个位置相对于通过 `background-origin` 定义的背景层的原点进行定位
</Mcard>

<Mcard>

### [`background-repeat`][zh-link]{ .mdn-link linkparam=background-repeat}
定义背景图像的重复方式。背景图像可以沿着水平轴，垂直轴，两个轴重复，或者根本不重复
</Mcard>

<Mcard>

### [`background-size`][zh-link]{ .mdn-link linkparam=background-size}
设置背景图片大小
</Mcard>

<Mcard>

### [`<basic-shape>`][zh-link]{ .mdn-link linkparam=basic-shape}
表现基础图形的 CSS 数据类型，作用于 `clip-path` 与 `shape-outside` 属性中
</Mcard>

<Mcard>

### [`::before(:before)`][zh-link]{ .mdn-link linkparam=::before}
伪元素，匹配选中的元素的第一个子元素。常通过 `content` 属性来为一个元素添加修饰性的内容。此元素默认为行内元素
</Mcard>

<Mcard>

### [`:blank`][zh-link]{ .mdn-link linkparam=:blank}
伪类，选择用户输入为空的输入框
</Mcard>

<Mcard>

### [`<blend-mode>`][zh-link]{ .mdn-link linkparam=blend-mode}
CSS 数据类型，用于描述当元素重叠时，颜色应当如何呈现。它被用于 `background-blend-mode` 和 `mix-blend-mode` 属性
</Mcard>

<Mcard>

### [`block-size`][zh-link]{ .mdn-link linkparam=block-size}
根据元素的书写模式定义了元素块的横向或纵向尺寸
</Mcard>

<Mcard>
