# K

<Mcard>

### [`khz`][zh-link]{ .mdn-link linkparam=frequency#khz}
CSS 数据类型，表示频率维度，例如语音的音高。目前它未在任何 CSS 属性中被使用
</Mcard>

<Mcard>

### [`@keyframes`][zh-link]{ .mdn-link linkparam=@keyframes}
CSS at-rule，通过在动画序列中定义关键帧（或 waypoints）的样式来控制 CSS 动画序列中的中间步骤
</Mcard>
