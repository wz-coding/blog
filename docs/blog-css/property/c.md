# C

#### 以 C 为首的属性

<Mcard>

### [`calc()`][zh-link]{ .mdn-link linkparam=calc()}
CSS 函数，允许在声明 CSS 属性值时执行一些计算
</Mcard>

<Mcard>

### [`caption-side`][zh-link]{ .mdn-link linkparam=caption-side}
将表格的标题 `<caption>` 放到规定的位置。但是具体显示的位置与表格的 `writing-mode` 属性值有关
</Mcard>

<Mcard>

### [`caret-color`][zh-link]{ .mdn-link linkparam=caret-color}
定义插入光标 `caret` 的颜色
</Mcard>

<Mcard>

### [`@charset`][zh-link]{ .mdn-link linkparam=@charset}
指定样式表中使用的字符编码。它必须是样式表中的第一个元素，而前面不得有任何字符
</Mcard>

<Mcard>

### [`:checked`][zh-link]{ .mdn-link linkparam=:checked}
CSS 伪类，表示任何处于选中状态的 `radio`, `checkbox` 或 `select` 元素中的 `option` 元素
</Mcard>

<Mcard>

### [`circle()`][zh-link]{ .mdn-link linkparam=basic-shape#circle()}
CSS 函数，表现基础图形的 CSS 数据类型，作用于clip-path 与 shape-outside 属性中
</Mcard>

<Mcard>

### [`clamp()`][zh-link]{ .mdn-link linkparam=clamp()}
CSS 函数，接收三个参数：最小值、首选值、最大值，把一个值限制在一个上限和下限之间，当这个值超过最小值和最大值的范围时，在最小值和最大值之间选择一个值使用
</Mcard>

<Mcard>

### [`clear`][zh-link]{ .mdn-link linkparam=clear}
指定一个元素是否必须移动 (清除浮动后) 到在它之前的浮动元素下面
</Mcard>

<Mcard>

### [`clip-path`][zh-link]{ .mdn-link linkparam=clip-path}
使用裁剪方式创建元素的可显示区域。区域内的部分显示，区域外的隐藏
</Mcard>

<Mcard>

### [`<color>`][zh-link]{ .mdn-link linkparam=color_value}
CSS 数据类型，`<color>` 表示一种标准 RGB 色彩空间（sRGB color space）的颜色
</Mcard>

<Mcard>

### [`color`][zh-link]{ .mdn-link linkparam=color}
设置颜色值的前景色以及文本装饰，并设置 `currentcolor` 值
</Mcard>

<Mcard>

### [`color-scheme`][zh-link]{ .mdn-link linkparam=color-scheme}
设置元素展示的主题颜色方案
</Mcard>

<Mcard>

### [`column-count`][zh-link]{ .mdn-link linkparam=column-count}
描述元素的列数
</Mcard>

<Mcard>

### [`column-fill`][zh-link]{ .mdn-link linkparam=column-fill}
控制内容如何划分为列
</Mcard>

<Mcard>

### [`column-gap`][zh-link]{ .mdn-link linkparam=column-gap}
设置元素列之间的间隔 `gutter` 大小
</Mcard>

<Mcard>

### [`column-rule`][zh-link]{ .mdn-link linkparam=column-rule}
CSS 简写属性，可以在多列布局中设定分割线的宽度、样式和颜色
</Mcard>

<Mcard>

### [`column-rule-color`][zh-link]{ .mdn-link linkparam=column-rule-color}
设置在多列布局中被画在两列之间的规则（线条）的颜色
</Mcard>

<Mcard>

### [`column-rule-style`][zh-link]{ .mdn-link linkparam=column-rule-style}
设置在多列布局中被画在两列之间的规则（线条）的样式
</Mcard>

<Mcard>

### [`column-rule-width`][zh-link]{ .mdn-link linkparam=column-rule-width}
设置在多列布局中被画在两列之间的规则（线条）的宽度
</Mcard>

<Mcard>

### [`column-span`][zh-link]{ .mdn-link linkparam=column-span}
设置元素跨越所有的列
</Mcard>

<Mcard>

### [`column-width`][en-link]{ .mdn-link linkparam=column-width}
在多列布局中设置理想的列宽
</Mcard>

<Mcard>

### [`columns`][zh-link]{ .mdn-link linkparam=columns}
CSS 简写属性，设置元素的列宽和列数
</Mcard>

<Mcard>

### [`conic-gradient()`][en-link]{ .mdn-link linkparam=conic-gradient()}
CSS 函数，创建一个由渐变组成的图像，颜色过渡围绕中心点旋转
</Mcard>

<Mcard>

### [`contain`][zh-link]{ .mdn-link linkparam=contain}
标示了元素及其内容尽可能独立于文档树的其余部分
</Mcard>

<Mcard>

### [`contain-intrinsic-block-size`][zh-link]{ .mdn-link linkparam=contain-intrinsic-block-size}
定义了元素受尺寸局限时浏览器用于布局的元素块向尺寸
</Mcard>

<Mcard>

### [`contain-intrinsic-height`][zh-link]{ .mdn-link linkparam=contain-intrinsic-height}
定义了元素受尺寸局限时浏览器用于布局的元素高度
</Mcard>

<Mcard>

### [`contain-intrinsic-inline-size`][zh-link]{ .mdn-link linkparam=contain-intrinsic-inline-size}
定义了元素受尺寸局限时浏览器用于布局的元素行向尺寸
</Mcard>

<Mcard>

### [`contain-intrinsic-size`][zh-link]{ .mdn-link linkparam=contain-intrinsic-size}
定义了元素受尺寸局限时浏览器用于布局的元素尺寸
</Mcard>

<Mcard>

### [`contain-intrinsic-width`][zh-link]{ .mdn-link linkparam=contain-intrinsic-width}
定义了元素受尺寸局限时浏览器用于布局的元素宽度
</Mcard>

<Mcard>

### [`content`][zh-link]{ .mdn-link linkparam=content}
用于在元素的 `::before` 和 `::after` 伪元素中插入内容
</Mcard>

<Mcard>

### [`content-visibility`][zh-link]{ .mdn-link linkparam=content-visibility}
控制元素是否渲染其内容，以及施加一组强局限
</Mcard>

<Mcard>

### [`contrast()`][zh-link]{ .mdn-link linkparam=filter-function/contrast}
CSS 函数，调整输入图像的对比度
</Mcard>

<Mcard>

### [`cos()`][zh-link]{ .mdn-link linkparam=cos()}
CSS 函数，返回某数的余弦值，此值介于 -1 和 1 之间。此函数含有单个计算式，此式须将参数结果按弧度数解析为 `<number>` 或 `<angle>`
</Mcard>

<Mcard>

### [`counter()`][zh-link]{ .mdn-link linkparam=counter()}
CSS 函数，返回一个代表计数器的当前值的字符串
</Mcard>

<Mcard>

### [`counter-increment`][zh-link]{ .mdn-link linkparam=counter-increment}
用于将 CSS 计数器 的值增加给定值
</Mcard>

<Mcard>

### [`counter-reset`][zh-link]{ .mdn-link linkparam=counter-reset}
用于将 CSS 计数器 重置为制定值
</Mcard>

<Mcard>

### [`counter-set`][en-link]{ .mdn-link linkparam=counter-set}
将 CSS 计数器 设置为给定值
</Mcard>

<Mcard>

### [`@counter-style`][zh-link]{ .mdn-link linkparam=@counter-style}
CSS `@` 语法 ，它让开发者可以自定义 counter 的样式
</Mcard>

<Mcard>

### [`counters()`][zh-link]{ .mdn-link linkparam=counters()}
CSS 函数，嵌套计数器，返回表示指定计数器当前值的连接字符串
</Mcard>

<Mcard>

### [`cross-fade()`][en-link]{ .mdn-link linkparam=cross-fade()}
CSS 函数，可用于以定义的透明度混合两个或多个图像
</Mcard>

<Mcard>

### [`cubic-bezier()`][zh-link]{ .mdn-link linkparam=easing-function}
CSS 数据类型，表示描述数值变化速率的数学函数
</Mcard>

<Mcard>

### [`::cue (:cue)`][zh-link]{ .mdn-link linkparam=::cue}
CSS 伪元素，匹配所选元素中的 `WebVTT` 提示，例如 `<video>`
</Mcard>

<Mcard>

### [`::cue-region`][en-link]{ .mdn-link linkparam=::cue-region}
CSS 伪元素，匹配所选元素中的 `WebVTT` 提示
</Mcard>

<Mcard>

### [`:current`][en-link]{ .mdn-link linkparam=:current}
CSS 伪类，表示一个时间维度的伪类，代表当前显示的元素或元素的祖先。例如，此伪类可用于表示 `WebVTT` 正在显示的带有字幕的视频
</Mcard>

<Mcard>

### [`cursor`][zh-link]{ .mdn-link linkparam=cursor}
设置光标的类型（如果有），在鼠标指针悬停在元素上时显示相应样式
</Mcard>

<Mcard>

### [`<custom-ident>`][zh-link]{ .mdn-link linkparam=custom-ident}
CSS 数据类型，指用户自定义字符串标识符，要区分大小写，值不能有任何歧义
</Mcard>

<Mcard>

### [`cap`][zh-link]{ .mdn-link linkparam=length#cap}
CSS 数据类型，表示元素字体 `font` 的 “上限高度”
</Mcard>

<Mcard>

### [`ch`][zh-link]{ .mdn-link linkparam=length#ch}
CSS 数据类型，代表元素所用字体 font 中“0”这一字形的宽度
</Mcard>

<Mcard>

### [`cm`][zh-link]{ .mdn-link linkparam=length#cm}
CSS 数据类型，厘米。 1cm = 96px / 2.54
</Mcard>
