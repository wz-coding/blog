# Z

#### 以 Z 为首的属性


<Mcard>

### [`z-index`][zh-link]{ .mdn-link linkparam=z-index}
设定了一个定位元素及其后代元素或 `flex` 项目的 z-order。当元素之间重叠的时候，z-index 较大的元素会覆盖较小的元素在上层进行显示（`position` 属性值不是 `static` 才生效）
</Mcard>
