                                                                  以 J 为首的 CSS 属性(样式)

                                                                  # J

                                                                  # 以 J 为首的属性

                                                                  # justify-content

                                                                  定义了浏览器之间，如何分配顺着弹性容器主轴 (或者网格行轴) 的元素之间及其周围的空间

                                                                  # justify-items

                                                                  为所有盒中的项目定义了默认的 justify-self，可以使这些项目以默认方式沿适当轴线对齐到每个盒子

                                                                  # justify-self

                                                                  设置单个盒子在其布局容器中的对其方式

                                                                  # justify-tracks

                                                                  在其内联轴上具有砌体的网格容器设置砌体轴的对齐方式
